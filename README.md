# Order System

## Get started

 1. Prerequisites
      - ruby 2.7.2
      - rails 6.1.0
      - PostgreSQL 10+
      - yarn

2. Regular rails setup
  ```bash
  $ bundle install
  $ rails db:create
  $ rails db:migrate
  $ rails db:seed
  $ yarn install
  ```
3. Start server
  ```bash
  $ rails server
  ```

4. runl test
  ```bash
  $ rails test
  ```

5. Default admin account
  ```bash
  email:     test@test.com
  password:  password
  ```
